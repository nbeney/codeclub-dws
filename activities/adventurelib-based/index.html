Build an adventure game with Adventurelib

Read this first

Adventurelib is a Python library for creating text-based adventure games.

First check that adventurelib is installed on your PC by running this program. If all is ok, the program will print OK, otherwise you will see an error message.

import sys sys.path.append('...') # PATH TO BE DETERMINED! from adventurelib import when, start, Bag, Item bag = Bag() @when("pick THING") def pick(thing): bag.add(Item(thing)) print(f"You have {len(bag)} items in your bag now.") @when("check") def check_bag(): if len(bag) == 0: print(f"Your bag is empty!") else: print(f"You have {', '.join(str(item) for item in bag)}") @when("throw THING") def throw(thing): if thing in bag: bag.take(thing) if len(bag) == 0: print(f"Your bag is empty now!") else: print(f"You have {len(bag)} items left in your bag.") else: print(f"Sorry, you have no {thing} in your bag!") start()

An interaction with the program could look like this:

> go north I don't understand 'go north'. > help Here is a list of the commands you can give: ? check help pick THING quit throw THING > pick apple You have 1 items in your bag now. > pick banana You have 2 items in your bag now. > check You have apple, banana > throw apple You have 1 items left in your bag. > throw egg Sorry, you have no egg in your bag! > check You have banana > quit
