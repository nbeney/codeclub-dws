What will I make?

A program to draw a flag from a country of your choice. Here is a selection of interesting flags but you can find many more at flagpedia.net.

Instructions

  1. Open the starter project and click the Run button.
  2. Login to be able to save your work.
  3. Check W3Schools to find color codes.
  4. Avoid using hard-coded numbers for coordinates and dimensions. Instead use simple calculations based on width and height. For example if the canvas is 300x200 and you need to draw a circle of diameter 50 in the center, use circle(width/2, height/2, 50) instead if circle(150, 100, 50). This way your drawing will adjust automatically if the size of the canvas changes.
