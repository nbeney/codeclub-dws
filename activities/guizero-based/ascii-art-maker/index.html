ASCII Art Maker

What will I make?

An application to convert text into ASCII Art fonts.

The GUI is comprised of a single window () with these widgets:

  • an image () for the header
  • a text field () to input the text to be converted
  • a drop down list () to select the desired font
  • a "color" button () to select the desired color
  • a "surprise" button () to select a random font
  • a multiline text field () to output the result

Instructions

  1. Download the starter kit which contains all the resources (eg images) you need and some code to get you started.
  2. Open the main.py file in your favourite editor (eg IDLE).
  3. Run the program. A window should pop up but nothing will happen when you enter some text or click the buttons. This is because the code is incomplete but you will fix this next.
  4. Modify main.py, adding the missing code in the places marked with # TODO. In particular:
    • Implement the callback function for when the text or font changes. Get the text from the input field and the font from the drop down list. Call the figlet_format function from the pyfiglet module with the text and the font as arguments to get the result. Put the result in the output field.

What can I try next?

  • Add a button to copy the output text to the clipboard.
  • Add a check box "Automatic" which is initially unchecked. When it is checked, the application will choose a random font every 10 seconds.
